# 🇯🇵 名古屋 & 高山專屬旅遊 網頁

這是一個專為 2026 年 5 月日本之旅設計的專屬 網頁，採用**大地色系 (Earth Tones)** 設計風格。

## 🌟 核心功能
- **行程管理**：5/19 - 5/23 精確行程，內建 Google Maps 導航連結。
- **必買推薦**：按區域（榮商圈、高山老街、AEON）整理的購物清單。
- **購物工具**：具備勾選功能的採買清單與備註。
- **實用工具**：即時匯率換算（JPY ➔ TWD）與 Google 翻譯連結。

## 🎨 設計規範
- 主色調：`#8C6A5D` (深可可)
- 背景：`#F4F2EE` (燕麥米)
- 點綴：`#626F47` (苔蘚綠)

## 🚀 如何運行
1. 電腦安裝 Node.js。
2. 下載專案：`git clone [你的專案網址]`。
3. 安裝零件：`npm install`。
4. 啟動 App：`npx expo start`。
5. 使用手機 **Expo Go** 掃描 QR Code 即可進入 App。
